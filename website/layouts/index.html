Kubernetes Custom Resources

Use Kubernetes Custom Resources to deploy and manage Thanos Receive, Query, Query Frontend, Store Gateway, Compactor and Ruler. Provides simple and easy to understand configuration.

Gradually Adoptable

This operator is designed to follow the same pattern of adoption as Thanos. Choose what components you need and would like to manage, and seamlessly integrate them into your current setup. Go from bare-bones simple setups to global scale monitoring!

Upstream Friendly

Built to be compatible with the latest Thanos releases, and to enhance and complement the Prometheus Operator. Thanos Operator tries to follow similar configuration styles and ensures both operators can be used in tandem.
